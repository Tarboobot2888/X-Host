# X-Host Cosmic System

## تشغيل المشروع

```bash
npm install
npm run dev
```

أو باستخدام pnpm:

```bash
pnpm install
pnpm dev
```

## جودة المؤثرات (Quality Tiers)

يتم اختيار الجودة تلقائيًا اعتمادًا على أداء الجهاز وتفضيلات تقليل الحركة:

- **low**: أقل عدد من الجسيمات، تقليل الظلال والمؤثرات الثقيلة.
- **medium**: توازن بين الأداء والجودة.
- **high**: كثافة جسيمات أعلى ومؤثرات كاملة.

عند تفعيل `prefers-reduced-motion` يتم فرض مستوى **low** لتقليل الحركة.

## مفاتيح التخزين (Storage Keys)

- `x-host-intro-seen:v2` (sessionStorage): يتذكر مشاهدة المقدمة السينمائية.
- `x-host-ui` (localStorage): الثيم، اللغة، تفضيلات الصوت.
- `x-host-selection` (localStorage): مسار الخدمة المختارة، الباقة، ووقت آخر اختيار.
